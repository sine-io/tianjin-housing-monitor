# Tianjin Housing Monitor

Bootstrap workspace for a free-first Tianjin housing monitor.

## Current Scope

Task 1 only establishes the shared TypeScript workspace, frozen community and segment config, and the config loader contract used by later collection and build tasks.

## Scripts

- `npm run test`
- `npm run typecheck`
- `npm run collect`
- `npm run promote:manual`
- `npm run build:data`
- `npm run build:site`
- `npm run build`
- `npm run test:e2e`
- `npm run preview`
